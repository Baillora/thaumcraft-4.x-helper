# Thaumcraft 4 Research Helper

This is a web application to assist with Thaumcraft 4.x research mechanics. The app provides tools for:
- Finding optimal paths to connect two aspects in research notes.
- Calculating Energized CV output for Aura Nodes.
- Calculating target CV for specific nodes.
- Simulating Hungry Node feeding and growth.
- A fully interactive Aspect Lexicon to view recipes.

## Hosting on GitHub Pages

You can host this project on GitHub Pages so you don't need to open the local files every time and can share it.

1. **Create a GitHub Repository**: 
   - Go to GitHub and sign in or create an account.
   - Click to create a **New** repository.

2. **Upload the Code**:
   - Upload all the project files (including `index.html`, and `css`, `js`, `aspects` directories) to your repository.

3. **Enable GitHub Pages**:
   - In your repository, click on the **Settings** tab.
   - On the left sidebar menu, click on **Pages**.
   - Under the "Build and deployment" section -> "Source", select the `main` or `master` branch. Click **Save**.
   
4. **Access the Web App**:
   - GitHub will provide you with a URL (e.g., `https://baillora.github.io/thaumcraft-4.x-helper/`).
   - Wait a minute or two, and your site will be live at the provided URL! Any future changes will be automatically updated.
