# Thaumcraft 4 Research Helper

This is a web application to assist with Thaumcraft 4.x research mechanics. The app provides tools for:
- Finding optimal paths to connect two aspects in research notes.
- Calculating Energized CV output for Aura Nodes.
- Calculating target CV for specific nodes.
- Simulating Hungry Node feeding and growth.
- A fully interactive Aspect Lexicon to view recipes.
